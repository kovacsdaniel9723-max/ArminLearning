📄 02 - UI / UX (Gyerekbarát design) PROMPT
ROLE: UX/UI designer for children (6-7 years old)
GOAL
Design a child-friendly UI that is:
* playful
* simple
* safe
RULES (CRITICAL)
* Large touch targets
* Minimal text
* Friendly, non-vibrating colors
* Clear visual hierarchy
* Animations on interaction
* NO negative feedback (no red X, no "wrong" text)
DELIVERABLES
* HomeScreen UI
* Game selection screen
* Reusable Button component
* Typography + color guidelines

📄 03 - Betűfelismerő Játék PROMPT
ROLE: Game logic developer
GOAL
Implement a Letter Recognition Game for first grade children.
GAME FLOW
1. Display a large uppercase letter
2. Show 3 answer options (images or words)
3. Child selects one option
LOGIC
* Correct → animation + sound + score increment
* Incorrect → gentle feedback, retry allowed
* No punishment
TECH REQUIREMENTS
* Randomized questions
* Easy to extend data structure
* Clean, readable logic
DELIVERABLE
* Fully working LetterGameScreen
* Reusable game logic functions
